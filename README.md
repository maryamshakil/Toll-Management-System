# 🚗 Toll Management System

## Overview

The **Toll Management System** is a desktop-based application developed using **Python (Tkinter)** and **Oracle Database**. It automates toll collection and management through separate **User** and **Admin** portals.

The system allows users to manage their vehicles, RFID tags, wallet balance, and toll history, while administrators can manage users, vehicles, RFID tags, toll plazas, motorways, transactions, reports, and feedback.

## Features

### User Portal

* Secure Login and Registration
* Profile Management
* Vehicle Management
* RFID Tag Management
* Wallet Recharge
* Toll Transaction History
* Payment and Recharge History
* Alerts and Status
* Feedback System

### Admin Portal

* Admin Dashboard
* User Management
* Vehicle Management
* RFID Tag Management
* Toll Plaza and Motorway Management
* Transaction Monitoring
* Reports and Analytics
* Feedback Management
* Admin Audit Log
* Camera Management
